# Content

| 快捷鍵            |         功能         | 備註          |
|:----------------- |:--------------------:|:------------- |
| Ctrl + O          |     快速切換筆記     |               |
| Ctrl + Alt + 左右 |    前一頁、後一頁    |               |
| Ctrl + E          |    預覽/編輯切換     |               |
| Ctrl + ,          |       打開設定       |               |
| Ctrl + K          |       插入連結       |               |
| Ctrl + B          |         粗體         |               |
| Ctrl + I          |         斜體         |               |
| Alt + V           |    貼圖傳到github    | 自訂          |
| Alt + T           |       插入樣板       | 自訂          |
| Alt + N           | 顯示筆記所在選單位置 | 自訂          |
| Alt + S           |  顯示筆記所系統位置  | 自訂          |
| Alt + G           |   顯示Local Graph    |               |
| Alt + F           |   全螢幕Focus Mode   | 自訂，esc退出 |
| Alt + E           |   在筆記內插入様板   | 自訂          |
| Alt + M           |       移動檔案       | 自訂          |
